# Quick Start - Sample Checkout Page

## Access the Checkout Demo

After deploying the fintech solution on Windows 11, you can access the sample checkout page at:

```
http://localhost:3000/checkout.html
```

Or if deployed on a different host:
```
http://your-server-ip:3000/checkout.html
```

## Quick Test

The checkout page is fully functional and ready to test:

1. **No setup required** - The page automatically generates a demo authentication token
2. **All payment methods available** - UPI, Cards, Net Banking, Wallets, QR, BNPL, EMI, Biometric
3. **Real API integration** - All forms connect to the backend APIs
4. **Instant feedback** - See API responses in real-time
